# ⚡ Quick Start Guide

Get started in **3 minutes**!

## 1️⃣ Install Dependencies
```bash
npm install
```
**Takes ~2 minutes** depending on your internet

## 2️⃣ Start Development Server
```bash
npm run dev
```

## 3️⃣ Open in Browser
```
http://localhost:3000
```

## ✅ You're Ready!

Try both flows:
- **Click "Manual Quiz"** for the form-based experience
- **Click "AI Voice Quiz"** to talk to the AI (requires microphone access)

## 🎤 Testing Voice

1. Click "AI Voice Quiz"
2. Allow microphone access when prompted
3. Listen to the AI's first question
4. Speak your answer clearly
5. Watch it get transcribed automatically!

## 📦 Build for Production

```bash
npm run build
npm start
```

## 🌐 Deploy to Vercel

```bash
npm install -g vercel
vercel
```

Answer the prompts and your app is live!

## 🆘 Troubleshooting

**Microphone not working?**
- Check browser permissions
- Try Chrome instead of Safari
- Use manual quiz as fallback

**Port 3000 already in use?**
```bash
npm run dev -- -p 3001
```

**Build errors?**
```bash
rm -rf .next node_modules
npm install
npm run dev
```
